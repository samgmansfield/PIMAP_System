PIMAP (Pervasive Injury Monitoring And Prevention)

License:
Author: Sam Mansfield

PIMAP is a system framework consisting of four components: Sense, Store, Analyze, and
Visualize. PIMAP uses two data-types to pass information PIMAP samples and PIMAP metrics.

Directories:
  pimap: Contains the core pimap components.
  tests: Contains the unittests for the core pimap components.

Files:
  requirements.txt: The requirements file used for the command:
    pip install -r requirements.txt
